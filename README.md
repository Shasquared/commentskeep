# CommentsKeep
> Never lose a great comment again.

A Chrome extension to bookmark, tag, and search YouTube comments.

Click the bookmark icon next to the Reply button on any YouTube comment to save it.

## Install
1. [Download the zip](./commentkeep.zip)
2. Unzip it
3. Open Chrome, Vivaldi, Opera, or any chromium based browser and go to `chrome://extensions`
4. Enable **Developer mode** (top right toggle)
5. Click **Load unpacked** and select the unzipped folder
6. Go to any YouTube video, scroll to comments, click the bookmark icon
   
## Features
- Save any YouTube comment in one click
- Captures the exact video timestamp
- Search across all saved comments
- Add tags to organise and filter
- Copy comment to clipboard
- Jump back to the exact video moment
